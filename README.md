# cyber_project
CyberDefenseX – ML and Blockchain powered automated SIEM and SOAR platform.
CyberDefence-X is an advanced cybersecurity solution designed to detect, analyze, and respond to cyber threats in real-time. It integrates SIEM (Security Information and Event Management) and SOAR (Security Orchestration, Automation, and Response) capabilities with modern technologies like Machine Learning, Blockchain, and Cloud Computing.

The platform aims to automate threat detection, improve incident response time, and enhance overall organizational security posture.
Features-

🔍 Real-Time Threat Detection
Monitors logs and network activities to identify suspicious patterns.

🤖 Machine Learning-Based Anomaly Detection
Detects unknown and zero-day attacks using intelligent models.

🔗 Blockchain-Based Log Integrity
Ensures tamper-proof storage of security logs.

⚙️ Automated Incident Response (SOAR)
Automatically responds to threats (e.g., blocking IPs, alerts).

☁️ Cloud-Based Scalability
Handles large-scale data efficiently using cloud infrastructure.

📊 Dashboard & Visualization
Interactive dashboards for monitoring and analytics.
